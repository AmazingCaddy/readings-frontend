# readings-frontend

前端开发进阶学习资料，面向有开发经验、希望系统补齐前端深度并准备中高级前端面试的学习场景。

站点使用 Docusaurus 构建，内容使用 Markdown/MDX，支持 Mermaid 图表，并准备通过 GitHub Pages 发布。

## 本地开发

```bash
npm install
npm run start
```

## 构建验证

```bash
npm run build
```

## 内容标准

每篇正式文章应覆盖：场景、是什么、为什么需要、推荐做法、代码示例、反例与后果、常见坑、排查与验证、面试怎么讲、延伸阅读。

详见 `docs/standards/article-template.md`。
