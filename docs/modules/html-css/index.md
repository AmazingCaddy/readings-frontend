# HTML & CSS 基础能力

本模块关注页面结构、布局、样式系统、响应式和可访问性。目标不是背 CSS 属性，而是能解释布局机制、视觉稳定性和工程化样式边界。

## 计划文章

- [CSS 布局、BFC、Flex 和 Grid](./layout-bfc.md)。
- [HTML 语义化、表单和可访问性](./semantic-accessibility.md)。
- CSS 层叠、优先级、继承和样式隔离。
- 响应式设计、容器查询和移动端适配。

## 面试目标

能把布局问题讲清楚：为什么元素塌陷、为什么 margin 合并、什么时候用 Flex/Grid、如何避免布局抖动，以及语义化如何影响可访问性和 SEO。
